# Reachy Mini Event Assistant App

Forked from the Reachy Mini conversation app.

Use the `src/reachy_mini_event_assistant_app/profiles/_reachy_mini_event_assistant_app_locked_profile` folder to customize your own app from this template:
- Edit instructions `_reachy_mini_event_assistant_app_locked_profile/instructions.txt`
- Edit available tools in `_reachy_mini_event_assistant_app_locked_profile/tools.txt`
- You can create your own tools in `_reachy_mini_event_assistant_app_locked_profile` by subclassing the `Tool` class.

Do not forget to customize:
- this `README.md` file
- the `index.html` file (Hugging Face Spaces landing page)
- the `src/reachy_mini_event_assistant_app/static/index.html` (the web app parameters page)

The original README from the conversation app is available in `README_OLD.md`.
